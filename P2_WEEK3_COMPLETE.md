## 🐛 Troubleshooting

### Issue: Worker not starting
**Solution:** Check Render.com dashboard - worker service should show as "Live"

### Issue: Jobs not running
**Solution:** Verify REDIS_URL is set and worker process is running

### Issue: Can't access /sidekiq
**Solution:** Only available in production. Use SIDEKIQ_USERNAME/PASSWORD from env vars

### Issue: Cache not refreshing
**Solution:** Check worker logs for errors. Verify Reddit API credentials.
